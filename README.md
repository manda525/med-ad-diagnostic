# 医療広告リスク診断ツール v10

薬剤師 × 医療広告コンサルタント「まさ」が開発する、薬機法・景表法・医療広告GLに基づくAI一次診断ツール。

## v10での変更点（2026-06-08）

- ルールブックをJSXから`rulebook.json`に独立分離
- 491件（v9: 487件 + 4件追加）
- 追加ルール：こむら返り予防、妊娠線抑制、腹囲表現、体験談形式の医薬品的効能（妊娠案件由来）

## デプロイ

### 必須環境変数（Vercel側で設定）

| 変数名 | 値 |
|---|---|
| `ANTHROPIC_API_KEY` | Anthropic Consoleで発行したAPIキー |

## ルール追加運用フロー

新しいNG表現を発見したらClaudeに自然言語で説明する。Claudeがrulebook.jsonの最新版に新ルールを追記して出力するので、それをGitHubにアップロード。Vercelが自動で再デプロイする。Excelを開く必要は基本的にない（年次監査やバックアップ時のみ）。

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。
